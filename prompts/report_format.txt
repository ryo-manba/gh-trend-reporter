以下の WeeklyAnalysis データを使って、GitHub Trending 週次レポートを Markdown 形式で生成してください。

## フォーマット仕様

### 1. YAML Frontmatter
レポートの先頭に YAML frontmatter を含めてください:
- week: 週ラベル（例: "2025-W03"）
- period: 期間（例: "2025-01-13 〜 2025-01-19"）
- generated_at: 生成日時（ISO 8601）
- model: 使用モデル名
- total_repos: 収集リポジトリ総数

### 2. タイトル
# GitHub Trending 週次レポート — {week_label}

### 3. 今週のハイライト
highlights リストを箇条書きで表示。

### 4. 言語別ランキング
top_languages を Markdown テーブルで表示:
| 順位 | 言語 | リポジトリ数 | 割合 |

### 5. カテゴリ別分析
categories の各カテゴリを ### セクションとして表示。
各カテゴリには summary_ja と注目リポジトリのテーブルを含める。

### 6. 新登場リポジトリ
new_entries を Markdown テーブルで表示。

### 7. 先週との比較
week_over_week テキストを表示。前週データがない場合はスキップ。

### 8. フッター
---
*Generated by gh-trend-reporter*
